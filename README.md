## What's This About?

This is my personal portfolio website built with React. It's basically my digital resume where people can see my projects, learn about my skills, and get in touch with me. I built it to have a professional online presence and share my work with potential employers and collaborators.

## Cool Things You'll Find Here

- **Smooth Animations** - Things move and fade in nicely when you scroll
- **Works on Any Device** - Looks great whether you're on your phone, tablet, or computer
- **Typing Effect** - My job titles type themselves out on the homepage (pretty cool, right?)
- **My Projects** - All my best work organized in one place
- **Contact Form** - You can send me a message directly from the website
- **Newsletter Signup** - Want updates? You can subscribe!

## What I Built This With

Don't worry if these sound technical - here's what powers the website:

- **React** - The main framework I used to build everything
- **Bootstrap** - Helps make the website look good and work on all screen sizes
- **CSS** - For all the colors, layouts, and styling
- **Node.js** - Powers the contact form so you can email me

## Want to Run This on Your Computer?

### What You Need First

- Node.js installed on your computer (it's free - just Google "download Node.js")
- That's pretty much it!

### Getting Started

1. **Download this project**
```bash
git clone https://github.com/yourusername/portfolio.git
cd portfolio
```

2. **Install everything it needs**
```bash
npm install
```
(This might take a minute - it's downloading all the tools the website needs)

3. **Start it up!**
```bash
npm start
```

Your browser should automatically open to `http://localhost:3000` and you'll see the website running!
